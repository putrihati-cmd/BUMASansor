# User Manual Warung (POS)

Panduan cepat untuk pengguna warung.

## 1. Instalasi

1. Minta file APK ke admin.
2. Install APK di perangkat Android.
3. Aktifkan izin kamera jika diminta (untuk scan barcode).

## 2. Login

1. Buka aplikasi BUMAS Ansor.
2. Masukkan email dan password dari admin.
3. Tekan tombol **Masuk**.

## 3. Penjualan Harian

1. Buka menu POS.
2. Tambahkan produk:
   - Scan barcode, atau
   - Pilih produk manual.
3. Atur jumlah item.
4. Pilih metode pembayaran:
   - Cash
   - Transfer
   - QRIS
   - EDC
5. Simpan transaksi.

## 4. Lihat Struk

Setelah transaksi sukses:

- Struk muncul otomatis.
- Anda bisa:
  - Copy isi struk.
  - Share struk.

## 5. Riwayat Penjualan

1. Buka menu **Riwayat Penjualan**.
2. Pilih transaksi untuk melihat detail item, total, dan pembayaran.

## 6. Cek Stok

1. Buka halaman utama warung.
2. Lihat daftar produk dan jumlah stok.
3. Jika stok kritis, hubungi admin/gudang untuk restock.

## 7. Piutang

1. Buka menu pembayaran/tagihan (jika tersedia pada role Anda).
2. Cek total sisa tagihan.
3. Lakukan pembayaran ke admin/kurir.

## 8. Mode Offline

- Jika internet terputus, transaksi tetap tersimpan lokal.
- Saat koneksi kembali, sistem akan sinkronisasi otomatis.
- Jangan uninstall aplikasi sebelum sinkronisasi selesai.

## 9. Tips Operasional

- Sinkronkan aplikasi setiap akhir hari (pastikan internet aktif).
- Cek tanggal/jam perangkat agar data transaksi akurat.
- Simpan bukti transfer jika pembayaran non-tunai.

## 10. Bantuan

Jika ada kendala:

1. Catat pesan error.
2. Screenshot halaman error.
3. Kirim ke admin support BUMAS Ansor.

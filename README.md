# BUMAS Ansor

Project ini sudah di-hard reset ke baseline minimum Next.js.

## Menjalankan project
1. Install dependency: `npm install`
2. Jalankan dev server: `npm run dev`

## Catatan
- Seluruh kode legacy, dokumentasi lama, integrasi lama, dan artefak deployment lama sudah dibersihkan.
- Gunakan ini sebagai titik awal untuk arsitektur baru.
